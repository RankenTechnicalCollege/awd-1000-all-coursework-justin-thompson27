Working with Positions

Sticky

This is an example of an element with a sticky position. Its position is based on the user's scroll position.

The sticky element can be used for information that the website should always have on the screen such as the company's logo.

Relative

This is an example of an element with a relative position. It can move around a little, depending on the viewport, but it is in a position relative to itself.

Fixed

This is an example of an element with a fixed position. Its position is relative to the viewport.

I changed the top value to 200px and the right value to 40px. The box is now stuck in place according the the viewport.

Absolute

This is an example of an element with an absolute position. It is similar to the fixed position; however, it is positioned relative to the closest parent element.

I used 79x for the bottom value and 40px for the right value. When the value was 0 for bottom the box was stuck at the bottom right of the screen. Now that I have changed the values it has moved slightly from there but is still mainly in a fixed position relative to the viewport.
